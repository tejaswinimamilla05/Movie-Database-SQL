# Movie Database Analysis (SQL)

## Description
This project focuses on analyzing a movie database using SQL queries to extract meaningful insights.

## Tools Used
- SQL (MySQL)

## Work Done
- Designed and explored movie database tables
- Wrote SQL queries using SELECT, WHERE, JOIN, GROUP BY, HAVING, ORDER BY
- Analyzed movies based on genre, ratings, revenue, and release year

## Key Insights
- Identified top-rated movies
- Analyzed genre-wise movie performance
- Extracted insights useful for recommendation and business analysis
